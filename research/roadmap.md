# Contextix Research Roadmap

Status: Initial

Research supports product development but does not block it.

## Research Priorities

1. Context Fidelity
2. Compression Quality
3. Cross-LLM Consistency
4. Project Understanding Benchmark
5. Knowledge Graph IR
6. Reasoning Preservation

## Initial Questions

### Context Fidelity

How much project understanding survives after memory generation and compression?

Initial metrics:

- Architecture retention
- Constraint retention
- Decision retention
- Task retention

### Compression Quality

How much can repository and document context be reduced without damaging understanding?

Initial metrics:

- Token reduction
- Duplicate removal rate
- Human review score

### Cross-LLM Consistency

Can different AI assistants use the same `.context/` output and reach similar conclusions?

Initial metrics:

- Agreement score
- Missing-context incidents
- Incorrect assumption count

## Near-Term Research Output

- Small reference repositories
- Golden `.context/` examples
- Manual evaluation checklist
- Baseline heuristic output comparison

## Principle

Research should produce clear evidence or useful negative results. It should not add product complexity until the value is demonstrated.
